# EasyLinalg

## To install

Julia> ] add https://github.com/pevifol/Easylinalg

or

using Pkg
Pkg.add("https://github.com/pevifol/Easylinalg")

## To develop

active the package.

Julia> cd("EasyLinal")

Julia> ] activate .

then you can use: using EasyLinalg


## To Build

Julia> ] build
